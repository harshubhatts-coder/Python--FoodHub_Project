# FoodHub - Restaurant Demand Analysis
## 📌 Project Overview
This project focuses on exploratory data analysis (EDA) for a food 
aggregator company. The goal was to understand customer demand 
patterns across different restaurants and cuisines, helping the 
business improve customer experience and make informed decisions.
## 🎯 Objective
- Understand which cuisines and restaurants are most in demand
- Analyze customer ordering behavior and delivery patterns
- Provide actionable business recommendations based on data insights
## 🛠️ Tools & Skills Used
- Python
- Pandas
- NumPy
- Seaborn
- Univariate Analysis
- Bivariate Analysis
- Exploratory Data Analysis (EDA)
